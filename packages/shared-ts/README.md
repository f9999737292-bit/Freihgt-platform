# @freight-platform/shared-ts

Shared TypeScript types and utilities used across freight platform services and applications.

## Exports

| Subpath | Description |
|---------|-------------|
| `@freight-platform/shared-ts/types` | Common domain and API types |
| `@freight-platform/shared-ts/utils` | Shared helper functions |

## Usage

```typescript
import type { AppHealthResponse } from "@freight-platform/shared-ts/types";
import { formatTimestamp } from "@freight-platform/shared-ts/utils";
```

## Development

```bash
npm run typecheck
```
