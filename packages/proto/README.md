# Protobuf Definitions

This package holds Protocol Buffer (`.proto`) definitions for gRPC services and event payloads in the freight platform.

## Location

Protobuf files live in the `definitions/` directory. Organize by domain or service (for example, `definitions/transport/`, `definitions/events/`).
