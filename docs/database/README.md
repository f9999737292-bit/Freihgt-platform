# Database Documentation

Database schemas, migrations, and data model references for the freight platform.

## Migrations

Schema changes are managed under `infrastructure/migrations`.

## Schemas

| Schema | Purpose |
|--------|---------|
| `core` | Core platform entities and shared reference data |
| `transport` | Shipments, routes, carriers, and logistics operations |
| `rfx` | Requests for quotation and bidding workflows |
| `documents` | Document metadata and storage references |
| `billing` | Invoicing, rates, and financial records |

Add detailed ER diagrams and table-level documentation in follow-up docs under this directory.
